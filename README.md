# landing-page-hero-neon
Landing page moderna com seção hero de alto impacto, fundo escuro, efeito glassmorphism no formulário de captura, botões com brilho neon azul, botão flutuante do WhatsApp e design totalmente responsivo. Desenvolvido com Next.js, Tailwind CSS e TypeScript.
# Landing Page Hero - Glassmorphism & Neon

Uma landing page moderna e impactante com design futurista, desenvolvida com as melhores práticas de desenvolvimento web.

## Funcionalidades

- Seção hero com fundo escuro e título impactante
- Formulário de captura com efeito glassmorphism (vidro fosco)
- Botões com brilho neon azul animado
- Botão flutuante do WhatsApp para contato rápido
- Rodapé com links para redes sociais
- Design totalmente responsivo (mobile, tablet e desktop)
- Favicon personalizado

## Tecnologias Utilizadas

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React (ícones)

## Como Executar

1. Clone o repositório
git clone https://github.com/tatiane347/landing-page-hero-neon.git

2. Instale as dependências
pnpm install

3. Execute o projeto
pnpm dev

4. Acesse no navegador
http://localhost:3000
